BMW E28 Tacho Enhancement(Instrument Cluster)

Hey folks,

in the middle of my BMW E28 turbo build project I am struggling with the idea how to display some important informations about the engine.

I am planning to use a free programmable ViPec V88 ECU. The advantage is, that I can get all the informations, which the ECU get's out from it via CAN Bus.

Therefore I am planning to use a SaintSmart Mega 2560 Controller Board - or compatible Arduino Board to read all the Infos from CAN Bus via a CAN Bus Shield wich is attached to the Controller Board. In the ViPec ECU Programming Tool, I can setup which data should be send from the ECU to the CAN Bus.

Maybe if you guys are planning to also modify your cockpit cluster, this will maybe be the difference, the source of the data to display.

Wishlist

  • Change ServiceIntervall Gauge to RPM Depending ShiftLight Example Idea
  • Use CheckControll Light to display general Warning which are send via the ECU
  • Switch functionality of the economy Gauge to boost pressure Example Idea

Check Control Light Control via Arduino/SaintSmart
